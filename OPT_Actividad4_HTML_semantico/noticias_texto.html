La importancia del HTML semántico

Usar etiquetas con significado ayuda a los buscadores, facilita la accesibilidad y mejora la compresion del codigo.

Por ejemplo, un lector de pantallaa entiende que un header, es una cabecera y no un contenedor visual

Buenas practicas de estructura

Usar encabezados en orden (h1,h2, etc.) permite una jerarquía clara en el contenido.

Notas del profesor

  • Repasar las etiquetas semánticas antes del examen.
  • Usar siempre estructura clara y jerarquía de títulos.
  • Comentar el código para mayor claridad.
